Settings

In the settings are all the parameters that change the simulation. You need to click "Apply settings" or press A for any changes you make to take effect.

Game Speed, Infection Radius, and Heal Radius apply instantly, the others apply when the next run is started.

Changing settings other than Simulation Speed or Sprite Size resets the statistics.
